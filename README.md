This is a lightweight demo starter for the Ground to Gather project.

- Next.js App Router structure (TypeScript)

- Mock API routes: /api/products, /api/payments
- Mock data in /data/*.json
- Logo and sample assets
- Included original project PDF for reference: `public/Ground_to_Gather_Full_6page_Report.pdf`

## How to run locally (demo, not published)
1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open http://localhost:3000

## Notes
- Authentication is a simple demo stub (no real auth). Use pages as demo.
- Payments endpoint is a mock. It always returns success.
- To quickly preview without installing, upload this project to CodeSandbox or StackBlitz (Next.js template).
